<h1 align="center">Python Project Maker</h1>

Run `bash pyproject.sh` in the desired directory to create a new project.

It will create a python project with the following file structure:

```text
project
│   .gitignore
│   LICENSE
│   MANIFEST.in
│   pyproject.toml
│   README.md
│   requirements.txt
│   setup.cfg
│   setup.py
│
└───project
    │   __init.py
    │   project.py
```
